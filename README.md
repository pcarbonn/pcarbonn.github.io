# pcarbonn.github.io

Website for [100-year-agenda.com](https://www.100-year-agenda.com).  Also hosts redirections, e.g., of pcarbonn.github.io/XMT-IDE.

## Development

This project uses **Tailwind CSS v4** for styling.

### Prerequisites

- [Node.js](https://nodejs.org/) (v20+)
- [npm](https://www.npmjs.com/)

### Setup

Install dependencies:

```bash
npm install
```

### Build

To build the CSS once:

```bash
npm run build
```

To watch for changes and rebuild automatically during development:

```bash
npm run watch
```

The output CSS is generated in `docs/output.css` and linked in `docs/index.html`.
